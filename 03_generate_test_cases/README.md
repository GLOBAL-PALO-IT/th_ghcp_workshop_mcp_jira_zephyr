# 04: Generate Test Cases from Jira/Confluence

## สร้าง Test Case อัตโนมัติด้วย Copilot

1. เปิด Jira Epic/Story หรือ Confluence ที่มี Requirement
2. ใช้ Copilot ช่วยวิเคราะห์และแนะนำ Test Case
3. คัดลอก Test Case ที่ได้ไปสร้างใน Zephyr MCP
4. ตรวจสอบและปรับแต่ง Test Case ให้เหมาะสม
