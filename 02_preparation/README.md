# 02: Preparation

## เตรียมความพร้อมก่อนเริ่มใช้งาน

1. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
2. เตรียมบัญชี Jira และสิทธิ์การเข้าถึง Zephyr
3. เตรียม API Token จาก Jira
   - ไปที่ Jira > Account Settings > Security > Create and copy API Token

4. เตรียมข้อมูล Epic/Story/Confluence ที่ต้องการสร้าง Test Case

---

## ตัวอย่างคำสั่งที่ใช้บ่อย
```bash
zephyr-mcp login --token <API_TOKEN>
```
> ใช้ล็อกอินด้วย API Token
